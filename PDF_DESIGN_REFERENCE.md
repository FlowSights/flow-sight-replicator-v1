- **Cada paso incluye:**

  - Consejos/Tips
  - Enlaces directos cuando aplica

## Elementos de Diseño Premium

1. **Colores:**
   - Emerald (#10B981) como color principal
   - Negro (#000000) para fondos oscuros
   - Blanco (#FFFFFF) para fondos claros
   - Gris oscuro para texto secundario

2. **Tipografía:**
   - Títulos: Bold, tamaño 24-32pt
   - Subtítulos: Bold, tamaño 14-18pt
   - Cuerpo: Regular, tamaño 10-12pt
   - Monoespaciado para enlaces

3. **Espaciado:**
   - Márgenes: 20mm en todos los lados
   - Espaciado entre secciones: 15-20mm
   - Padding interno de cards: 10-15mm

4. **Elementos visuales:**
   - Líneas separadoras emerald (ancho 3-4pt)
   - Cards con bordes redondeados (4-6pt)
   - Badges con esquinas redondeadas
   - Sombras sutiles en cards
   - Fondos oscuros para destacar información

## Mejoras a Implementar

1. **Integración de Mockups Visuales:**
   - Renderizar previsualizaciones de cada anuncio por plataforma
   - Incluir imagen del anuncio en contexto

2. **Métricas Dinámicas:**
   - Calcular proyecciones basadas en presupuesto y plataforma
   - Mostrar CPC, CPM, ROI estimado

3. **Múltiples Anuncios:**
   - Mostrar todos los anuncios generados (Oferta, Emocional, Urgencia)
   - Incluir razonamiento de cada anuncio

4. **Guías Personalizadas:**
   - Adaptar pasos según la plataforma seleccionada
   - Incluir enlaces directos a la plataforma

5. **Entregables:**
   - PDF profesional y descargable
   - Opción de exportar a otros formatos
